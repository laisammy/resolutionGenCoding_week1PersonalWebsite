# Portfolio Website
## Made for Hack Club's Resolution General Coding Week 1
This project is a personal portfolio website built with React, Vite, and Tailwind CSS. It includes multiple pages such as Home, Projects, and Contact, all connected through a responsive navigation bar made with React Router. The site showcases some personal information about me, coding projects, and a fun cat image generator powered by TheCatAPI.

Great opportunity to learn about vite, react and tailwind!

You can visit the website [here](https://resolution-general-coding-week1.vercel.app/)
